#  Darpit’s Color Design

This project is a creative animation made with **HTML, CSS, and JavaScript**.  
It displays **beautiful colorful circles** that move and bounce across the screen —  
like a digital art piece created with code! 

 **Features**
- Dynamic motion & color effects  
- 100% made by Darpit  
- Pure HTML, CSS & JS (no libraries!)  

 **Perfect For**
- Beginners exploring creative coding  
- Learning canvas animations  
- Adding visual art to web pages
